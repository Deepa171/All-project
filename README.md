# 🎵 Spotify Clone

A fully responsive front-end clone of Spotify, built with modern web technologies to mimic the design and layout of the real Spotify web player.

## 🚀 Features

- 🎧 Homepage with playlist cards
- 🔍 Search functionality UI
- 📻 Player UI with controls (play/pause, skip, etc.)
- 📱 Fully responsive design for desktop & mobile
- 🌙 Dark theme inspired by the real Spotify interface

## 🛠️ Tech Stack

- **HTML5** – Semantic and structured layout
- **CSS3** – Custom styles and responsiveness
- **JavaScript** – Interactive UI elements
- **Bootstrap / Tailwind** – For rapid styling (if used)
